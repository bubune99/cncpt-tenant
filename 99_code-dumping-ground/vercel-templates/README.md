# Vercel Templates

## Available Templates

### 1. platforms-main
Vercel's Platforms Starter Kit - A template for building multi-tenant applications with custom domains.

**Useful Components:**
- Multi-tenant architecture
- Custom domain management
- Site creation workflow
- Dashboard layouts
- Authentication flows

**Integration Notes:**
- Extract multi-tenant patterns
- Adapt dashboard components
- Use domain management logic

### 2. multi-tenant-starter-template-main
Multi-tenant SaaS starter template with comprehensive features.

**Useful Components:**
- Tenant isolation patterns
- Subscription management
- User role management
- API structure
- Database schemas

**Integration Notes:**
- Study tenant isolation approach
- Extract subscription logic
- Adapt role management system

## Extraction Workflow

1. **Analyze Structure**: Review the template's architecture
2. **Identify Components**: Mark useful components for extraction
3. **Document Patterns**: Note architectural patterns to adopt
4. **Extract Code**: Copy relevant code to appropriate folders
5. **Add Meta-tags**: Enhance with our meta-tag system
6. **Test Integration**: Verify in development branches
