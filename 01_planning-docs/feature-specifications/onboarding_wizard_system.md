## VIDEO-GATED FEATURE PROGRESSION SYSTEM

### Training-Based Feature Unlocking

#### Video Training Requirements
```sql
feature_training_requirements {
  id: UUID PRIMARY KEY
  feature_name: VARCHAR(255)
  plugin_name: VARCHAR(255) NULL
  required_video_id: UUID FOREIGN KEY
  minimum_watch_percentage: INTEGER DEFAULT 80
  quiz_required: BOOLEAN DEFAULT FALSE
  quiz_passing_score: INTEGER NULL
  prerequisite_features: JSONB NULL
  user_level_requirement: ENUM (beginner, intermediate, advanced, expert)
  estimated_training_time: INTEGER (minutes)
  feature_complexity_score: INTEGER (1-10)
  business_impact_level: ENUM (low, medium, high, critical)
  created_at: TIMESTAMP
}
```

#### User Training Progress
```sql
user_training_progress {
  user_id: UUID FOREIGN KEY
  video_id: UUID FOREIGN KEY
  watch_percentage: DECIMAL(5,2) DEFAULT 0
  completed_at: TIMESTAMP NULL
  quiz_score: INTEGER NULL
  quiz_attempts: INTEGER DEFAULT 0
  notes_taken: TEXT NULL
  bookmarks: JSONB NULL
  feature_unlocked: BOOLEAN DEFAULT FALSE
  unlock_date: TIMESTAMP NULL
  last_watched: TIMESTAMP
  total_watch_time: INTEGER DEFAULT 0 (seconds)
  PRIMARY KEY (user_id, video_id)
}
```

#### Feature Access Control
```sql
user_feature_access {
  user_id: UUID FOREIGN KEY
  feature_name: VARCHAR(255)
  plugin_name: VARCHAR(255) NULL
  access_granted: BOOLEAN DEFAULT FALSE
  granted_at: TIMESTAMP NULL
  granted_by: ENUM (training, admin_override, subscription_tier, beta_access)
  access_level: ENUM (view_only, basic, full, admin)
  expires_at: TIMESTAMP NULL
  usage_count: INTEGER DEFAULT 0
  last_used: TIMESTAMP NULL
  PRIMARY KEY (user_id, feature_name, plugin_name)
}
```
